## Phase 1：基礎固め（3月7日〜4月12日）

**目標**：N1全範囲の語彙・文法を一通り把握する

### 語彙
- [ ] 「新完全マスター語彙 N1」1周目（1日20語 × 5週 = 700語）
- [ ] 漢字読みカード作成（問題1対策）
- [ ] 類義語グループ表を作る（問題3対策）

### 文法
- [ ] 「新完全マスター文法 N1」1周目（1日4ページ）
- [ ] N1文法179形式を一覧表で整理
- [ ] 各形式に例文1つ暗記

### 読解
- [ ] 日本語の論説・エッセイを毎日1本読む（NHKウェブ記事・朝日新聞）
- [ ] 読後に50字で要約する練習

### 聴解
- [ ] NHKラジオ「まいにち中国語」ではなくNHK「聞いてしゃべる」等で耳慣らし
- [ ] Podcast「ゆる言語学ラジオ」などで自然な日本語に毎日30分接触

## Phase 2：集中インプット（4月13日〜5月31日）

**目標**：弱点セクションの重点補強、問題形式の完全習熟

### 語彙（問題2・4重点）
- [ ] 「新完全マスター語彙 N1」2周目（弱点語のみ）
- [ ] 用法問題（問題4）の例文200文を暗記
- [ ] カタカナ語・慣用句・ことわざ100個を仕上げる

### 文法（問題5・6重点）
- [ ] 「新完全マスター文法 N1」2周目（ニュアンス差を比較しながら）
- [ ] 紛らわしい文法ペアを30セット整理
  - 例：〜にかかわらず vs 〜をよそに vs 〜をものともせず
- [ ] 並べ替え問題（問題6）を毎日5問解く

### 読解（問題9・10・12重点）
- [ ] 中文・長文を毎日1本解いて解説を精読
- [ ] 統合理解（問題11）の比較読み練習
- [ ] 解答時間の計測を始める（問題10に15分以内の習慣化）

### 聴解（問題4重点）
- [ ] 即時応答表現200パターンの暗記
- [ ] 毎日聴解練習問題5問（「新完全マスター聴解 N1」）
- [ ] NHKニュースを聞いて要点メモ取り練習

## Phase 3：実戦演習（6月1日〜6月28日）

**目標**：本番形式での時間内完答 → 安定して100点超え

### 週1回フル模試（計4回）
| 回 | 実施日 | 使用教材 |
|----|--------|---------|
| 第1回 | 6月7日（日） | 公式過去問 2024年12月回 |
| 第2回 | 6月14日（日） | 公式過去問 2023年7月回 |
| 第3回 | 6月21日（日） | 公式過去問 2022年12月回 |
| 第4回 | 6月28日（日） | 模擬試験（「完全模試 N1」） |

### 模試後の復習ルール（当日中に必ず実施）
1. 全問の正誤確認（〇・△・×で分類）
2. × の問題：解説を読んで根本原因を特定
3. △ の問題：「なぜ迷ったか」をメモ
4. 間違えた語彙・文法を専用ノートに記録

### 平日：セクション別強化
- 月・水：読解問題集（1日大問1つ）
- 火・木：語彙・文法問題集（1日20問）
- 金：聴解（1日問題1〜4を通し練習）
- 土：弱点セクション集中補強
