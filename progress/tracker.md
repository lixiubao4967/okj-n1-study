| 〜ばこそ | 正因為…才…（強調唯一の理由、書き言葉） | 〜からこそ | 2026-06-02 |
| 〜からには | 既然…就（後接義務・決意） | 〜が早いか（一…就立刻…） | 2026-06-02 |
| 〜にして接続 | 「踏まえて」は「にして」と接続不可。形式に注意 | 踏まえて／下地にして | 2026-06-02 |
| 〜なり | 一…就立刻（同主語・意外な動作） | が最後／が早いか | 2026-06-02 |
| 〜以上は | 既然…就（＝からには、義務・決意） | ばかりに（就因為…导致坏结果） | 2026-06-02 |
| 〜が最後／たら最後 | 一旦…就（不可逆の悪い結果） | なり／次第 | 2026-06-02 |
| ★要復習：「一…就」5句型 + からには≠ばかりに（2回連続ミス） | 対比表をtracker参照 | | 2026-06-02 |
| 〜が最後／たら最後 | 一旦…就（不可逆の悪い結果）。信号「もう…できない」 | そばから（随…随…，反復） | 2026-06-03 |
