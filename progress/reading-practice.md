## 練習1：長文読解（問題12型・主張理解長文）

- 出題日：2026-06-04

### 本文

　「便利になることは、いいことだ」——私たちはほとんど無意識のうちに、そう信じて生きている。新しい道具やサービスが登場するたびに、それまで手間のかかっていた作業が一瞬で片づくようになり、私たちは浮いた時間をより創造的なことに使えるはずだ、と期待する。だが、その期待は本当に裏切られていないだろうか。

　たとえば、目的地までの道順を考えてみよう。かつて私たちは、地図を広げ、現在地を確かめ、いくつかの経路を頭の中で比較し、目印となる建物を覚えながら歩いた。それは確かに面倒な作業だった。しかし、その過程で私たちは、街全体の構造を自然と頭に描いていた。今では、画面の矢印に従って歩くだけで目的地に着く。早く、確実に。ところが、同じ道を翌日もう一度たどろうとすると、私たちはしばしば思い出せない。<u>①道は通ったのに、覚えていないのだ。</u>

　ここに、便利さがひそかに奪っているものの正体がある。手間とは、単なる無駄ではない。それは、私たちが対象と深く関わり、自分なりの理解を築くための時間だったのだ。手間を省くということは、その理解を築く機会そのものを省くことにほかならない。便利な道具は、結果だけを差し出す。だが、結果に至るまでの過程こそが、私たちの思考を鍛えていたのである。

　もちろん、すべての手間を礼賛しようというのではない。私たちの一日は有限であり、あらゆる作業を昔ながらのやり方で行えば、それだけで一日が終わってしまう。重要なのは、<u>②どの手間を手放し、どの手間を抱え続けるかを、自分で選び取ること</u>だ。ところが現実には、便利さはしばしば「選択」としてではなく、「初期設定」として私たちの前に現れる。気がつけば、考える前に答えが提示され、迷う前に道が示されている。私たちは選んでいるつもりで、実は選ばされているのかもしれない。

　便利さそのものが悪いのではない。問題は、便利さに身を委ねることが、いつのまにか「考えないこと」と同義になってしまう点にある。考えるという行為は、本来エネルギーを要するものだ。だからこそ人は、考えずに済む道があれば、自然とそちらへ流れていく。便利な道具は、その流れをどこまでも加速させる。

　では、どうすればよいのか。答えは逆説的だ。便利な時代だからこそ、私たちは時に、あえて不便を選ぶ必要がある。すべてを効率化するのではなく、自分にとって本当に大切な領域だけは、手間という名の時間を惜しまない。そこでこそ、私たちは再び、対象と深く関わる感覚を取り戻すことができる。<u>③便利さを使いこなすとは、便利さに使われないことなのである。</u>

### 設問

**問1**　下線部①「道は通ったのに、覚えていないのだ」とあるが、なぜか。

2. 画面に従うだけで、自分で道を理解する過程を経ていないから
3. 翌日には街の構造そのものが変わってしまっていたから
4. 目的地に早く着くことだけに気を取られていたから

**問2**　下線部②「どの手間を手放し、どの手間を抱え続けるかを、自分で選び取ること」について、筆者の考えに最も近いものはどれか。
1. できるかぎり多くの手間を省き、創造的な作業に時間を使うべきだ

3. 何を効率化し何を効率化しないかを、自分の判断で決めるべきだ
4. 手間を抱えるかどうかは、その日の時間の余裕に応じて決めるべきだ

**問3**　下線部③「便利さを使いこなすとは、便利さに使われないことなのである」とは、どういうことか。
1. 便利な道具の仕組みを完全に理解してから使うべきだということ
2. 便利さに流されず、自ら考え選ぶ姿勢を保つべきだということ
3. 便利な道具はできるだけ使わないようにすべきだということ
4. 便利さの恩恵を、他人より多く受け取るべきだということ

**問4**　この文章で筆者が最も言いたいことは何か。

2. 便利さは時間を生むが、その時間の使い方が問われている
3. 便利さは思考の機会を奪いうるため、意識的に不便を選ぶ姿勢が必要だ
4. 不便な生活こそが、人間本来の豊かな思考を育てる

### 答え合わせメモ（2026-06-06）

- 正解：問1=2／問2=3／問3=2／問4=3
- 問3ミスの原因：比喩句「便利さに使われない」（=被便利支配）を字面義「仕組みを理解する」で解釈。**选项在原文中无呼应句即为错**——"合理但文中没写"是典型陷阱。

## 練習2：綜合訓練#1（語彙复测＋比喩換言短文＋随筆読解）

### Part B 比喩句换言短文（B1✓ B2✗ B3✗）

1. 「ページをめくった手の記憶より、立ち止まった心の記憶こそが読書量」→ 正解：冊数より深く考えた経験（呼応：降りて歩いた町の数）
2. 「コンクリートに残された足跡」→ 正解：失敗の記録が消えずに残り続ける（呼応：記録され、いつまでも参照され続ける）。✗選んだ「深刻な結果」=合理但没写
3. 「濾過された水に、川の生態系は宿らない」→ 正解：単純化で現実の複雑さが失われる（呼応：複雑な現実を単純な図式に流し込む）。✗選んだ「理解力低下」=主語のすり替え（説明≠聞き手）

### Part C 随筆「写真館と待つ時間」（C1✓ C2✓ C3✓ C4✗）
