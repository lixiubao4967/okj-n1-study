# 日本語能力試験 N1 学習プロジェクト

## 試験概要

| 項目 | 内容 |
|------|------|
| 試験時間 | 言語知識・読解：110分 / 聴解：60分 |
| 合格基準点 | 総合100点以上（各セクション最低19点） |
| 満点 | 180点（各60点×3） |
| 試験日 | 年2回（7月第1日曜・12月第1日曜） |
| 2026年試験日 | 7月5日（日）/ 12月6日（日） |
| 第1回出願期間 | 3月17日（火）〜 4月7日（月） |

## セクション配点

```
言語知識（文字・語彙・文法） ── 60点
読解                         ── 60点
聴解                         ── 60点
─────────────────────────────────
合計                         ── 180点  合格ライン：100点
```

## ファイル構成

```
okj-n1-study/
├── README.md               # 本ファイル（概要・配点）
├── question-types.md       # 題型詳細解説（全問題形式）
├── study-plan.md           # 学習プラン（フェーズ別）
├── past-exams.md           # 過去問・教材リスト
├── daily-schedule.md       # 日次・週次スケジュール
└── progress/
    └── tracker.md          # 学習進捗記録
```

## クイックスタート

1. `question-types.md` で全題型を把握する
2. `study-plan.md` でフェーズを確認する
3. `daily-schedule.md` で今日のタスクを確認する
4. `past-exams.md` で演習教材を入手する
